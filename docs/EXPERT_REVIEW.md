# 三位专家讨论记录

## 1. 前端结构专家
- 旧版本把 CSS、JS、HTML 和 base64 图片混在同一个文件里，维护困难。
- 本次改为 `index.html + assets/css/style.css + assets/js/main.js`，结构更清楚。
- 重要区域都加了注释，后续改文章、改音乐、改颜色都更容易找。

## 2. 交互兼容专家
- 保留三端适配：手机、iPad、电脑。
- 顶部栏回到页面最顶部会自动收起，并带有动画。
- 音乐播放器支持弹窗、歌单滚动、上一首/下一首、顺序/列表循环/单曲循环。
- 调色板保留主题色相、背景模式、星星/流星/雾光开关，手机端标题不会竖排。

## 3. 部署与资源专家
- 所有资源文件名都改成英文 ASCII，避免 Render / GitHub / iPad Safari 出现乱码路径。
- 添加 `render.yaml` 和 README 部署说明。
- 音乐、图片、图标都放到清晰目录，上传部署时直接上传整包即可。
